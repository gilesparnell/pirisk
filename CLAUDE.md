# PiRisk — PiTime

## Project
Voice-activated timesheet & invoice app for PiRisk Management (Allerick).

## Stack
Next.js 15 | TypeScript | Turso (libSQL) | Drizzle ORM | NextAuth v5 | Tailwind CSS 4

## Local Dev
```bash
npm run dev    # http://localhost:3001 (NOT 3000 — avoids conflict)
npm run test   # vitest (147 tests)
```
